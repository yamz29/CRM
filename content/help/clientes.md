# Clientes

## ¿Qué hace?
Almacena el directorio de clientes del negocio: personas particulares y empresas con sus datos de contacto.

## ¿Para qué sirve?
Tener un registro centralizado de todos los clientes. Cada proyecto, presupuesto, factura y oportunidad se vincula a un cliente, lo que permite ver el historial completo de relación comercial.

---

## Cómo se usa

### Crear un cliente
1. Ir a **Clientes → Nuevo cliente**
2. Completar los datos:
   - **Nombre**: nombre completo o razón social
   - **RNC / Cédula**: para facturación fiscal (opcional)
   - **Tipo**: Particular o Empresa
   - **Teléfono / WhatsApp / Correo**: datos de contacto
   - **Dirección**: ubicación del cliente
   - **Fuente**: cómo llegó (Directo, Referido, Redes sociales, etc.)
3. Guardar

### Vista del cliente
Desde la ficha del cliente puedes ver:
- Proyectos asociados
- Presupuestos enviados
- Oportunidades en el pipeline
- Tareas pendientes

---

## Campos importantes

| Campo | Qué significa |
|-------|--------------|
| Tipo de cliente | Particular (persona física) o Empresa (persona jurídica) |
| RNC | Registro Nacional de Contribuyentes — necesario para comprobantes fiscales |
| Fuente | Canal de adquisición: Directo, Referido, Instagram, Web, etc. |
| WhatsApp | Número de WhatsApp para comunicación rápida |

---

## Tips

- Completa el RNC para clientes empresa — es necesario para las facturas con NCF
- Usa el campo "Fuente" consistentemente para saber qué canal trae más clientes
- No dupliques clientes; busca primero antes de crear uno nuevo
